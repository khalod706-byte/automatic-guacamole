# موقع "منيو السعادة"

موقع ويب ثابت (Static) صفحة واحدة بالاتجاه RTL مخصّص لعرض قائمة مطعم "منيو السعادة".

الملفات:
- `index.html` — الصفحة الرئيسية (HTML).
- `assets/styles.css` — CSS صغير لتعديلات خاصة.
- `assets/script.js` — تفاعلات بسيطة (قائمة الموبايل، تنظيف النموذج).
- (يمكن إضافة مجلد `images/` لصور مخصصة بدل روابط Unsplash).

التشغيل محلياً:
1. انسخ الملفات إلى مجلد.
2. افتح `index.html` في المتصفح (أو شغّل خادم محلي مثل: `npx http-server` أو `python -m http.server`).

نشر على GitHub Pages:
1. أنشئ مستودعًا جديدًا أو استخدم المستودع الموجود.
2. أضف الملفات إلى الفرع الرئيسي `main` (أو gh-pages).
3. في إعدادات المستودع (Settings) -> Pages اختر الفرع ومجلد الجذر `/` ثم اضغط Save.
4. انتظر لحين توفير الرابط (عادة دقائق).

اقتراحات تطوير مستقبلية:
- إضافة صفحة فرع/ساعات عمل باستخدام ملف JSON للبيانات لسهولة التحديث.
- استخدام Tailwind عبر بناء محلي (Tailwind CLI / PostCSS) لتقليل حجم CSS.
- استبدال action للـ form بخلفية (serverless function) أو استخدام خدمات مثل Formspree للتعامل مع الطلبات.
- إضافة صور حقيقية لكل صنف وصيغة responsive (srcset + lazy loading).
- تحسين SEO بتفاصيل JSON-LD لكل عنصر قائمة وأسعار منفصلة.

لو حابة، أقدر:
- أرفع هذه الملفات مباشرة إلى المستودع (أنشئ فرع وادفع التغييرات) — أخبريني إن أردتِ ذلك، وسأطلب منك تأكيد اسم المستودع/الفرع إذا لزم.
- أصمم نسخة مطبوعة (print CSS) أو قالب للطباعة.
- أضيف صفحة إدارة قائمة (واجهة بسيطة لتعديل العناصر) باستخدام Firebase أو Netlify CMS.
